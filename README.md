*Employee Leave Management System*
The Employee Leave Management System is a desktop-based application designed to simplify and automate the process of managing employee leave requests.

✨ Key Features
Employees can apply for different types of leave (casual, sick, earned, etc.).

Managers can review, approve, or reject leave requests.

Tracks leave balances and maintains accurate records.

Provides a structured workflow to reduce manual effort and paperwork.

Improves efficiency and transparency in HR processes.

🛠️ Tech Stack
Language: Java

Type: Desktop Application

📌 Purpose
This system helps organizations streamline leave management, ensuring smooth communication between employees, managers, and HR while maintaining compliance with company policies.
